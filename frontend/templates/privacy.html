Privacy Policy

Effective Date: Jun 11th, 2025

This web application (“the App”) is designed for planning and managing minister appointments in a game setting. It is an open-source tool intended to be self-hosted by users or alliances. This Privacy Policy explains how the App handles data.

1. Data Collected

  • Username – The in-game display name.
  • User ID – An internal identifier, usually tied to the game system.
  • State – The game state the user belongs to.
  • Alliance – The alliance name or tag the user is affiliated with.
  • Minister Appointments – Dates and times reserved for ministers.

2. Purpose of Data Collection

The App uses this data only for in-game organization and scheduling purposes. No personal or real-world identity information is collected.

3. Hosting and Data Control

This App is self-hosted. All data is stored on the server of the hosting user or group. The open-source developer does not collect, access, or transmit any user data.

4. GDPR Considerations

If deployed within the European Union or serving EU users, the host must ensure compliance with the General Data Protection Regulation (GDPR). This includes ensuring data is not misused, and that users can request data deletion or access.

5. Security

Although the App does not collect sensitive data, hosts are encouraged to implement standard security measures (e.g., HTTPS, access control) to protect data integrity and privacy.

6. User Rights

Users may request access, correction, or deletion of their data from the administrator of the hosted instance. The open-source project itself has no access to this data.

7. Developer Disclaimer

This is an open-source project. The developer does not run or store any version of the App and is not responsible for compliance of self-hosted instances. All responsibility lies with the host.
